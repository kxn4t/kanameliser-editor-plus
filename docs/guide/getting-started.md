# はじめに

## 必要環境

| 環境 | バージョン |
|---|---|
| Unity | 2022.3.22f1 以上 |
| NDMF（省略可） | — Mesh Info Display のビルドプレビュー機能に必要 |
| Modular Avatar（省略可） | 1.13.0 以上 — MA Material Helper 機能に必要 |

## インストール

### VRChat Creator Companion 経由（推奨）

1. [kxn4t.github.io/vpm-repos/](https://kxn4t.github.io/vpm-repos/) にアクセス
2. **Add to VCC**ボタンをクリックして「Kanameliser VPM Packages」リポジトリをVCC / ALCOMに追加
3. Manage Projectからパッケージ一覧上の**Kanameliser Editor Plus**をプロジェクトに追加

### 手動インストール

1. [GitHub Releases](https://github.com/kxn4t/kanameliser-editor-plus/releases) から最新リリースをダウンロード
2. ダウンロードしたパッケージをUnityプロジェクトにインポート
