# Missing BlendShape Inserter

アニメーションファイル間で不足している BlendShape キーを自動検出・補完します。すべてのアニメーションが同じ BlendShape を操作するようにファイルを更新することで、異なる表情状態間のスムーズな遷移を確保します。複数ファイルの一括処理にも対応しています。

## 主な使用場面

- アバターの顔の BlendShape を変更したが、表情アニメーションを更新していない
- ジェスチャーなどによる表情アニメーション切り替え時に表情が崩れる
- すべてのアニメーションファイルで一貫した BlendShape 操作を確保したい

## アクセス方法

`Tools > Kanameliser Editor Plus > Missing BlendShape Inserter`
