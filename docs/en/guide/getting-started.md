# Getting Started

## Requirements

| Requirement | Version |
|---|---|
| Unity | 2022.3.22f1 or higher |
| NDMF (optional) | Required for Mesh Info Display build preview |
| Modular Avatar (optional) | 1.13.0 or higher — required for MA Material Helper |

## Installation

### Via VRChat Creator Companion (Recommended)

1. Visit [kxn4t.github.io/vpm-repos/](https://kxn4t.github.io/vpm-repos/)
2. Click **Add to VCC** to add the "Kanameliser VPM Packages" repository to VCC or ALCOM
3. Add **Kanameliser Editor Plus** to your project from the package list in Manage Project

### Manual Installation

1. Download the latest release from [GitHub Releases](https://github.com/kxn4t/kanameliser-editor-plus/releases)
2. Import the downloaded package into your Unity project
